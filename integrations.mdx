## API Access for Custom Tools

Build bespoke integrations using our REST API. Generate an API key from your account settings, then make authenticated requests.

<Steps>
  <Step title="Generate API Key" icon="key">
    Navigate to Account > API Keys. Click "Create New Key" and copy the value securely.
  </Step>
  <Step title="Authenticate Requests" icon="shield">
    Add the key to your `Authorization` header as `Bearer {your_api_key}`.
  </Step>
  <Step title="Test Endpoint" icon="play">
    Call the orders endpoint to verify connectivity.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
  const response = await fetch('https://api.easyshoppingclub.com/v1/orders', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  });
  const orders = await response.json();
  console.log(orders);
  ```
  ```python
  import requests

  headers = {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
  }
  response = requests.get('https://api.easyshoppingclub.com/v1/orders', headers=headers)
  orders = response.json()
  print(orders)
  ```
  ```bash
  curl -H "Authorization: Bearer YOUR_API_KEY" \
       -H "Content-Type: application/json" \
       https://api.easyshoppingclub.com/v1/orders
  ```
</CodeGroup>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token obtained from your dashboard.
</ParamField>

<ParamField query="limit" param-type="integer" required="false">
  Number of records to return (default: 50, max: 1000).
</ParamField>

## Export and Import Options

Export customer data, orders, or products in multiple formats. Import supports bulk updates for efficiency.

<Tabs>
  <Tab title="CSV Export" icon="download">
    Download structured files for spreadsheets.
    
    ```
    Customer ID,Name,Email,Orders Count
    cust_123,John Doe,john@example.com,5
    cust_456,Jane Smith,jane@example.com,12
    ```
    
    <Callout kind="info">
      Exports include up to 10,000 rows. Use pagination for larger datasets.
    </Callout>
  </Tab>
  <Tab title="JSON Import" icon="upload">
    Upload JSON files to create or update records.
    
    ```json
    [
      {
        "email": "newuser@example.com",
        "name": "New User"
      }
    ]
    ```
  </Tab>
  <Tab title="API Bulk" icon="database">
    Use POST `/v1/customers/bulk` for programmatic imports.
  </Tab>
</Tabs>

<ExpandableGroup>
  <Expandable title="Advanced Export Filters" default-open="false">

  </Expandable>
</ExpandableGroup>
